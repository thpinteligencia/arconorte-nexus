## 🚀 Início Rápido

### Pré-requisitos
- Node.js 18+
- Python 3.9+
- Venv (Recomendado)

### Executando o Backend
```bash
cd api
python -m venv venv
source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
uvicorn main:app --reload
```

### Executando o Frontend
```bash
npm install
npm run dev
```
