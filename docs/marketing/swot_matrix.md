# Matriz SWOT: ArcoNorte Nexus

| **FORÇAS (STRENGTHS)** | **FRAQUEZAS (WEAKNESSES)** |
| :--- | :--- |
| • Modelos LSTM customizados para o Arco Norte | • Dependência de banco de dados SQLite (em migração) |
| • Métrica proprietária IPE (Pressão Logística) | • Necessidade de maior volume de dados históricos em certas UFs |
| • Interface UX intuitiva e moderna | • Equipe técnica enxuta para suporte 24/7 |
| • Painel de Auditoria para transparência total | • Custo de processamento de modelos de Deep Learning |

| **OPORTUNIDADES (OPPORTUNITIES)** | **AMEAÇAS (THREATS)** |
| :--- | :--- |
| • Expansão para Milho e Fertilizantes | • Alterações drásticas no layout do ComexStat/Governo |
| • Venda de APIs para Tradings e Bancos (AgroFintech) | • Entrada de grandes consultorias globais no setor |
| • Integração com sensores de IoT em Portos Secos | • Instabilidade em políticas de exportação agrícola |
| • Modelo Data-as-a-Service (DaaS) escalável | • Cyber-segurança (proteção de modelos proprietários) |
